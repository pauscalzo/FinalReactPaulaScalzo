# Descripción del Proyecto
Entrega Final del curso de React JS de CoderHouse, donde realizamos un e-commerce.
### Instalación
- git clone https://github.com/pauscalzo/FinalReactPaulaScalzo.git
- npm install
- npm run dev
### Tecnologías
- React JS
- Node
- Vite
- React Router Dom
- Firebase
- Bootstrap
- Sweet Alert 2
